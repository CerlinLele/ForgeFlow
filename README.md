# ForgeFlow

ForgeFlow is an early-stage manufacturing workshop product concept focused on real-time project progress, material stock, reorder requests, and customer visibility.

## Repository Structure

- `docs/`: research notes, raw chat logs, chat summaries, product thinking, market research, and business notes.

This repository is currently used as a product and research documentation space. Code repositories can be created separately later for web, API, mobile, or other implementation work.
